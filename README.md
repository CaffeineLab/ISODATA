# isodata

Your credential should look like this, and your PJM PKI Certificate needs to be supplied just as two file paths would be to a requests post or get.  More on this later.
```
{
  "user": "caffeinelab",
  "credentials":
  {
    "pjm":
    {
      "username": "***USERNAME***",
      "password": "***PASSWORD***"
    }
  }
}
```
